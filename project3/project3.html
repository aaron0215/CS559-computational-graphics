Description

In this project, I want to simulate a jumping Vileplume in a Gameboy screen. Gameboy is a
mobile game console released by Nintendo in 1989.

Along with Gameboy, Nintendo's "Pokemon Yellow" became one of the most phenomenal game
ever in the history. Vileplume is a Grass type Pokemon first appeared in Pokemon Yellow.

You can toggle "Vileplume View" above to see my Vileplume model.
